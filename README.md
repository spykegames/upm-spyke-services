# Spyke Services

Backend services UPM package for Spyke games.

## Features

- **Network** - WebSocket client, REST client, request queue
- **Auth** - Guest, Facebook, Apple authentication
- **Analytics** - Analytics tracking and events
- **Time** - Server time synchronization
- **Push** - Local notifications

## Installation

Add to your `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.spykegames.services": "https://github.com/spykegames/upm-spyke-services.git#v1.0.0"
  }
}
```

## Requirements

- Unity 6.3 LTS (6000.0)
- com.spykegames.core
